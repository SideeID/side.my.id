## The Breaking Point

Windows dan gw udah jalan bareng lama banget. Tapi belakangan ini, setiap kali nyalain laptop, rasanya kayak lagi lihat bloatware convention. Updates yang gabisa kamu tolak, RAM hilang entah kemana padahal cuma idle, updatenya ga habis habis dah.

Yang paling parah? Pas buka task manager, RAM usage kayak lagi di sale. "Ini ngapain sih?", terus Windows kayak "¯\\\_(ツ)\_/¯".

Tapi setelah bertahun-tahun lived in the terminal sebagai developer, gw pengen sesuatu yang **beneran dengerin** kebutuhanku.

## Gerah, Pengen Upgrade

Gini nih, udah lama banget jadi developer ngekos di terminal. Write code, manage servers, deploy apps, handle infrastructure semua terjadi di terminal window. Windows mulai terasa kayak training wheels pada saat itu.

gw pengen system yang **built for** orang kayak gw. Pengen customize everything, optimize setiap byte. Gaming? mainin beberapa game, tapi bukan priority. Programming adalah tempat 90% waktu gw. Jadi Windows punya "gaming advantage" tuh mulai terasa cuma bloat doang.

## The Leap: CachyOS + Hyprland

gw full masuk **Arch Linux**. Well, technically [CachyOS](https://cachyos.org/) basically Arch tapi udah di optimize dari awal buat performance. To be honest? First boot itu wow banget.

**Smooth as Hell**

gw pair-in sama [Hyprland](https://hyprland.org/) sebagai window manager (instead of GNOME atau KDE), terus throw in [Kitty](https://sw.kovidgoyal.net/kitty/) sebagai terminal. Kombinasinya _chef's kiss_ mamamia.

## First Impressions

RAM usage? RAM cuma 1.5GB total. Di Windows, itu miracle banget kalo bisa di bawah 8GB idle.

Boot time? Kita lagi ngomong sub-3 seconds dari power button sampe siap.

Desktop-nya responsive banget. Animasi smooth like butter. Aplikasi launch instant. Bukan berarti Linux itu lebih _cepat_ (meskipun bisa aja), tapi semua terasa _intentional_ nggak ada yang running di background.

Hyprland tiling window management yang berasa kayak punya personal assistant yang organize workspace. Kitty terminal-nya cepet banget dan customizable.

## Learning Curve? Totally Worth It

Ya, ada learning curve. gw harus setup own configurations, troubleshoot beberapa hardware stuff, terus get used to doing things "the Linux way". Tapi abis beberapa hari, nyambung.

Honestly? Dibanding Windows yang force through settings menus yang designed sama orang yang apparently never used a computer, Linux itu refreshingly straightforward. Mau change something? Edit config file. Done. Nggak ada GUI yang weird-weird buried 3 layer in. dan sebagai developer yang udah lama, itu exactly what I needed.

## Verdict gw

Recommend switch to Linux? **Absolutely.** Tapi nggak untuk semua orang sih. Kalo lu:

- Live di terminal (developers, sysadmins, DevOps folks)
- Udah capek sama bloatware dan resource waste
- Pengen full control terhadap system
- Nggak masalah buat ngeconfig-config
- Pake software yang available di Linux

Buat orang yang cuma mau buka Spotify dan browse Instagram? Windows probably fine. Tapi buat gw? Linux itu home sweet home.

gw udah pake beberapa minggu, and honestly nggak kepikiran buat balik lagi. Nggak ada forced updates. Nggak ada mysterious RAM consumption. Nggak ada bloatware. Just me, my code, dan system yang snappy.

**"I use Arch btw" bukan cuma meme it's a lifestyle choice.** 😎
