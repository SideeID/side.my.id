CF Warehouse is a specialized dataset storage platform designed for Politeknik Negeri Jember. It enables users to efficiently manage datasets by providing features to upload, view, and download data in multiple formats. The platform integrates Google Single Sign-On (SSO) for seamless authentication, ensuring that only authorized users with Politeknik Negeri Jember accounts can access the system.

## Key Features

- **Dataset Management:** Upload, view, and download datasets in various formats.
- **Google SSO Integration:** Secure login using Politeknik Negeri Jember Google accounts.
- **User-Friendly Interface:** Intuitive design for easy navigation and dataset management.
- **Secure Access Control:** Ensures that only authorized users can access and manage datasets.
