Kuruto is a static web page project created as an appreciation for Herta, my favorite character from Honkai Star Rail. This project features basic information about Herta, images, and simple interactive elements.

## Key Features

* Responsive design for various screen sizes.
* Image gallery showcasing Herta.
* Character information.
* "Squish the Kuru~" button with spinning Herta and sound effect.
* Click counter to track how many times you've "squished" Herta.

## Technologies Used

* HTML
* CSS
* JavaScript
