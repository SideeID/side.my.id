Logbook Cargo is a web application developed to streamline logistics data management at Adi Soemarmo Airport. It provides a robust platform for recording, tracking, and managing cargo logistics, enhancing operational efficiency.

## Key Features

- **Logistics Data Management:** Efficiently record, track, and manage logistics data.
- **User-Friendly Interface:** Intuitive design for ease of use.
- **Offline Access (PWA):** Progressive Web App (PWA) functionality for offline access and installation.
- **Data Tracking and Reporting:** Comprehensive tracking and reporting features.
- **Secure Authentication:** Secure user authentication to protect sensitive data.

## Technologies Used

- **Next.js:** React framework for building the application.
- **React:** For building the user interface.
- **Prisma:** ORM for database interactions.
- **PWA (Progressive Web App):** For offline access and installation.
- **Material UI:** For UI components.
- **Next Auth:** For authentication.
- **Dexie:** For local database.
- **Apexcharts & Recharts:** For data visualization.
