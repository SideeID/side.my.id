This application serves as an internal tool for the Ministry of Religious Affairs to monitor annual performance and streamline data and report management. It facilitates the input of work documents for employees across a 6-level user hierarchy.

## Key Features

* **Annual Performance Monitoring:** Tracks and monitors annual performance data.
* **Data and Report Management:** Simplifies the management of data and reports.
* **Work Document Input:** Enables employees to input work documents.
* **6-Level User Hierarchy:** Implements a 6-level user hierarchy for access control.
* **Document Generation:** Generates document in pdf and excel format.

## Technologies Used

* **Laravel:** A PHP web framework for building the application.
* **Excel/Spreadsheet Integration:** For handling spreadsheet data.
* **PDF Generation:** For generating PDF reports.
* **Dompdf:** For generate pdf document.
* **Maatwebsite/excel:** For generate excel document.

## Important Note

This application is for internal use by the Ministry of Religious Affairs, and due to confidentiality, detailed information cannot be disclosed.
