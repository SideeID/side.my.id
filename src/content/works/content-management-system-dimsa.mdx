This project is a Content Management System (CMS) tailored for Pondok Pesantren Darul Ihsan Muhammadiyah Sragen. It provides a robust platform for managing various aspects of the institution's content and administrative operations.

## Key Features

- **Content Management:** Efficiently manage and update website content, including articles, announcements, and events.
- **User Management:** Secure user authentication and role-based access control.
- **Administrative Tools:** Streamlined tools for managing administrative tasks.
- **Customizable Interface:** A user-friendly and customizable interface to meet the specific needs of the institution.
