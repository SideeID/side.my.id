Ezy API is a free REST API service for developers that actually makes sense. It gives you access to powerful LLMs, generates images, detects AI-written text, downloads media from social platforms, and handles file uploads. All without overcomplicating things or charging you a dime.

## Features

- **AI at your fingertips** – Access various LLMs for whatever you need to build
- **Create images** – Generate images from text descriptions
- **Know what's real** – Detect if text was written by AI
- **Download anything** – Media from Facebook, Instagram, YouTube, X, Threads, Xiaohongshu, all in one place
- **Upload with ease** – Simple file uploads without the headache
- **Docs that actually help** – Every endpoint explained, with examples that work
