Ezy API Documentation is a free REST API service designed for developers. It offers a range of services, including access to various Large Language Models (LLMs), image generation, AI word detection, social media media downloads, and file uploads.

## Key Features

* **LLM Access:** Provides access to various Large Language Models for natural language processing tasks.
* **Image Generation:** Generates images based on user prompts.
* **AI Word Detection:** Detects AI-generated text.
* **Social Media Downloads:** Downloads media from various social media platforms (Facebook, Instagram, YouTube, X, Threads, Rednote/Xiaohongshu).
* **File Uploads:** Uploads files to the API.
* **Comprehensive Documentation:** Provides detailed documentation for all API endpoints.

## Technologies Used

* **Next.js:** React framework for building the application.
* **Tailwind CSS:** For styling the application.
* **@radix-ui/\*:** UI components for accessibility.
* **framer-motion:** For animations.
* **lucide-react:** Icon library.
* **react-dropzone:** For file uploads.
* **tremor/react:** For dashboard component.
